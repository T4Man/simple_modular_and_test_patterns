###To run the greet function

1. Open a terminal window and navigate to the project root folder.
2. Enter 'npm install'
3. When complete enter 'node index.js [name]'
```
node index.js Tim
```

###To run mocha testing

1. Open a terminal window and navigate to the project root folder.
2. Enter 'mocha'
